==================  Mysql  ==================
Detail:  
https://www.w3schools.com/sql/

- For login mysql DB.
```
mysql -u root -p
```

- Show all the databases.
```
SHOW DATABASES;
```

- Select a DB and enter in.
```
use [DBname];
```

- Show all the tables in this DB.
```
show tables;
```

- Show this table data structure.
```
describe [tabeName];
```

- Logout the mysql.
```
exit
```
